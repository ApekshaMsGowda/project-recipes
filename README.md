# 🍽️ Foodie Recipes

A simple recipe website built using HTML. The website contains a collection of delicious recipes with detailed descriptions, ingredients, and step-by-step cooking instructions.

## Features

- Home page listing all recipes
- Individual recipe pages
- Navigation between pages
- Images for each recipe
- Ingredients list
- Cooking instructions
- Clean HTML structure

## Recipes Included

- 🍕 Pizza Margherita
- 🥧 Apple Pie
- 🥪 Halloumi, Arugula and Tomato Sandwich

## Technologies Used

- HTML5
